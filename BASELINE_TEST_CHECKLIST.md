# Baseline Test Checklist

Voer deze tests uit **VOOR** je begint met Fase 1. Test alle pagina's en documenteer eventuele problemen.

## Test Procedure per Pagina

### 1. Open pagina in browser
### 2. Open Developer Tools (F12)
### 3. Check Console tab voor errors/warnings
### 4. Test alle interactieve componenten
### 5. Noteer bevindingen
